🧠 Learn Mathematical Notation

As I said — we won’t do a lot of math.

But we will use notation. And it helps to get comfy with it.

👉 Don’t be scared off by Greek letters or funny symbols: \[ \text{logit}(p_i) = \beta_0 + \sum_{j=1}^k \beta_j x_{ij} \]

✍️ Some examples

\(x\) or \(y\) – a variable or value

\(\bar{x}\) – the mean (average) of \(x\)

\(n\) – number of observations

\(\sum\) – summation: e.g \(\sum_{i=1}^n x_i\) means “add all the \(x_i\) values from \(i = 1\) to \(n\)”

\(x_i\) – value of variable x for observation (row) i

\(x_{ij}\) – value in row i, column j of a data matrix

\(\widehat{y}\) – “y-hat” = estimated or predicted value

\(\alpha, \beta, \gamma, \delta, \varepsilon\)… – Greek letters we assign meaning to

\(\mathbb{E}[X]\) – expected value of \(X\)

\(f(x)\) – a function of x (a rule that maps inputs to outputs)

\(y = a + b x\) – a straight line

🧩 Why This Matters

  • Notation \(=\) universal language of math and stats
  • Helps you understand books, slides, articles, even ChatGPT explanations
  • Once you learn it, you stop seeing it as scary and start seeing it as helpful
